Weblog Management

Control Panel Location:  Admin > Weblog Administration > Weblog Management

This section of the Control Panel is for the management of weblogs. It is where weblogs are created, deleted, and preferences are set.

The main Weblog Management screen shows a table of all the existing weblogs. It lists the weblog's full name and provides links to edit the preferences.

  • Create a New Weblog: Enables you to create a new Weblog
  • Edit Preferences: Edit all the preferences for the weblog.
  • Edit Groups: Assign the category group, status group, and field group to use with that weblog.
  • Delete: Delete the weblog.
