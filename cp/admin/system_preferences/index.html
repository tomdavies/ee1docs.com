System Preferences

Control Panel Location:  Admin > System Preferences

The System Preferences section of the Control Panel allows you to set master preference and configuration options. There are several sections under System Preferences:

  • General Configuration
  • Control Panel Settings
  • Security and Session Preferences
  • Throttling Configuration
  • Localization Settings
  • Database Settings
  • Email Configuration
  • Mailing List Preferences
  • Image Resizing Preferences
  • Tracking Preferences
  • Cookie Settings
  • Captcha Preferences
  • Search Term Log Configuration
  • Word Censoring
  • Emoticon Preferences
  • Output and Debugging Preferences
