Control Panel Settings

Control Panel Location:  Admin > System Preferences > Control Panel Settings

This section of the Control Panel allows you to define the basic settings for your Control Panel.

Default Control Panel Theme

This setting specifies the default theme to use for the Control Panel. This is the theme that will be shown unless a user specifies another theme in their My Profile settings.

Publish Tab Behavior

This setting allows you to determine how the drop-down list on the Publish tab is triggered. The default setting of "Hover" will cause the list to appear whenever the mouse is hovered over the tab. You may alternatively select "Click", which will require you to click on the button before the list will appear, or "None", which will take you to a page with a list of available weblogs.

Default XML Language

This preference determines the XML language setting used for the Control Panel pages. A drop-down list is available for selection.

Default Character Set

This setting determines the character set used in the <meta> tag for Control Panel pages. A drop-down list is available for selection.
