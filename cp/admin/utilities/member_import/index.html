Member Import Utility

Control Panel Location:  Admin > Utilities > Import Utilities > Member Import Utility

The Member Import Utility enables you to import members from other systems utilizing ExpressionEngine's special XML format. It also includes a tool to convert delimited data into ExpressionEngine's Member XML Format.

  • Import from XML File
  • Convert Delimited Data to XML
