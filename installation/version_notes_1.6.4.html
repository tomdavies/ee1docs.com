Update Notes for Version 1.6.4

Comment Notification Template

The {name} variable has been removed from the User Comment Notification template. If you are still using this variable, you will need to instead use {name_of_recipient} for the recipient's name and {name_of_commenter} for the commenter's name.

You can find this template in Admin > Specialty Templates > User Comment Notification.
