Trackbacks not being sent

Trackbacks are not being sent from ExpressionEngine.

Troubleshooting Trackbacks

In order for Trackbacks to be sent out ensure that there is data in the field specified in the "Which field should be used for Trackbacks excerpt?" setting under Admin > Weblog Management. For instance:

  1. The weblog set to excerpt from the "Summary" field for Trackbacks.
  2. Create a new entry, but don't put any content in the "Summary" field.
  3. You specify a Trackback URL and submit your entry.

Because there is no data in the "Summary" field, ExpressionEngine can not send a Trackback out; there needs to be content in whatever field that has been specified for the excerpt.
