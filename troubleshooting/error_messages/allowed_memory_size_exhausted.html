Allowed memory size of ... bytes exhausted

ExpressionEngine returns the following error: "Allowed memory size of ... bytes exhausted."

Troubleshooting memory issues

This error indicates that a PHP process on the server ran out of memory. The amount of available memory for PHP will vary from server to server; while there are no minimal requirements for ExpressionEngine, a value substantially below 16 MB will frequently cause problems. Some processes, such as data import or export, Moblog email checking, file uploads, or image processing can also require much larger amounts of memory.

The solution to this error is to increase the memory limit. This would have to be done by the server administrator or host.
