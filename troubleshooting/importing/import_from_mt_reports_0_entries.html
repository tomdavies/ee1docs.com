MT Import says 0 entries imported

When attempting to import entries using the Movable Type Import Utility, the Utility returns "0 entries imported."

Troubleshooting the MT Import

Though it is acceptable in Moveable Type for the TITLE field to be optional, this field is required for files to work with the MT Import utility. Ensure that the import file contains the TITLE attribute.

Incorrect:

AUTHOR: Me
TITLE:
DATE: 02/18/2006 09:20:00 PM
-----
BODY:
...entry body...
-----
--------

Correct:

AUTHOR: Me
TITLE: My Title for This Entry
DATE: 02/18/2006 09:20:00 PM
-----
BODY:
...entry body...
-----
--------
