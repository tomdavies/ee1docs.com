Weblog Module

Note:  In our user guide and throughout the program you'll find the use of the word "weblog". If you are not using EE to manage a weblog then simply substitute the word "section" for it instead. A "weblog" in ExpressionEngine can be thought of as a "section" of your site.

ExpressionEngine's Weblog Module allows you to display content created using the PUBLISH page in the Control Panel.  The Weblog Module is the most powerful system within ExpressionEngine, allowing you a great deal of flexibility for displaying your content.

The Weblog Module provides the following tags and features:

  • Weblog Entries Tag
    • Parameters
    • Conditional Variables
    • Single Variables
    • Variable Pairs
    • Pagination
    • Spanning an Entry Across Multiple Pages
    • Entry "Views" Tracking
    • Dynamic Parameters
  • Weblog Information Tag
  • Next/Prev Entry Linking
  • Archive Month Links Tag
  • Calendar Tag
  • Weblog Categories Tag
  • Category Archive Tag
  • Category Heading Tag
  • Creating Relationships Between Entries
  • Displaying Reverse Relationships Between Entries
  • Stand-Alone Entry Form
