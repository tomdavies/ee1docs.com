Create a New Gallery

This feature is accessed through the Control Panel: Modules > Photo Gallery > Create a New Gallery. There are multiple steps involved in creating a new gallery.

Step One

You must first create the directory on your server that will hold the images you submit. The directory must be set to 777 file permissions or to be writable by the server.

Please submit the name of your new image folder

This is simply the name of the folder you created on your server. You may also specify the full server path to the directory you created for this gallery. A server path is typically something similar to /home/domain.com/http_docs/images/gallery/. The specific setting will vary from server to server so you may need to contact your Host or server admin to determine what your correct "server path" is.

Step Two

You must specify a name for the new Template Group that will be created for your photo gallery. You may not specify a name that corresponds to an already existing Template Group.

Step Three

The final step is to set the preferences for your new gallery. You may also access these preferences later.

Please see the Preferences section for information on the settings.
