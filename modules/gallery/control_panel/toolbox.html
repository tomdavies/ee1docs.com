Photo Gallery Toolbox

Once you have selected a Photo Gallery to access (Modules > Photo Gallery) you may choose the Toolbox menu option to access advanced tools that allow you to manipulate your existing images. You may crop or resize your image, rotate it, and even make copies of the image. These are advanced features that bring a great deal of power to your fingertips.

There are three main operations you can perform on your images:

  • Cropping
  • Resizing
  • Rotation

In addition, you may also specify to have the altered images saved as a new file as well as determine whether existing thumbnails will also be updated to match.

Select an Image to Edit

You must first select an image to edit. Your existing gallery images are listed in the drop-down. Simply select the desired image and ExpressionEngine will load that image in the Toolbox.

Enable Cropping

ExpressionEngine provides a real-time interface to allow you to crop any of your gallery images. To turn this feature on in the Toolbox, first simply check the "Enable Cropping" checkbox.

When active, a number of features become available:

  • Width: Specify the width (in pixels) of the cropped box.
  • Height: Specify the height (in pixels) of the cropped box.
  • Top: Specify the starting postion (in pixels from the top) of the cropped box.
  • Left: Specify the starting postion (in pixels from the left) of the cropped box.
  • Aspect Ratio: You may check a 4:3 or 1:1 aspect ratio and EE will constrain the box to the specify aspect ratio. You may select "none" to have no restraint.

You will notice that when you have the "Enable Cropping" feature active, a box appears over the image. This box represents the cropping area. You may move the box around by clicking and dragging with your mouse. You may alter the size of the box by holding down the SHIFT key while dragging the mouse; the box will resize visually for you. Any changes to the box position or size will be reflected in the options.

Enable Resizing

Resizing an image is simple to do with the Toolbox. First, you must enable the feature by checking the "Enable Resizing" checkbox.

When active, a number of features become available:

  • Width: Specify the width (in pixels) for the resized image.
  • Height: Specify the height (in pixels) for the resized image.
  • Constrain Proportions: This option allows you to ensure that the existing proportions of the image will be maintained when resized.

You may also dynamically resize the displayed image by by holding down the SHIFT key while dragging the mouse; the image will resize visually for you. Any changes to the box position or size will be reflected in the options.

Enable Rotation

You may select to have your image rotated by any of the available options:

  • 90° Left
  • 90° Right
  • 180°
  • Flip Vertical
  • Flip Horizontal

Options

Image Name

You may elect to have you altered image replace the existing image or you may specify a new filename and have a new image created (while retaining your original image).

Image Quality

You may also select the quality setting to use when saving the altered image. This is set as a value from 1 to 100 and indicates the quality setting to be used. Higher quality settings will result in better looking images but also in larger file sizes. Settings of "70" to "90" are typically used.

Apply Watermark

This allows you to choose whether or not to apply a watermark to the image. You can see the watermarking section for more information about setting up a watermark.

Create/Update Thumbnails

If you have existing thumbnails that were originally created from this image, you may select this checkbox to have the thumbnails automatically updated to reflect the changes you have made to the image. This will also update any "medium sized images" that were originally created from this image.

If you make changes to an existing image in your gallery, then you should go edit and entries associated with that image and Update them. This will make sure that ExpressionEngine picks up any changes you may have made to the image size.
